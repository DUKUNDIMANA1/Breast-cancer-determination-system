Gmail Setup Guide

  1. Go to Google Account → Security
  2. Enable 2-Step Verification
  3. Go to App Passwords
  4. Select Mail → Generate
  5. Copy the 16-character password
  6. Paste it in the App Password field above
Use your Gmail address as the email, and the 16-char App Password (not your Gmail login password). Free limit: 500 emails/day.

Brevo Setup Guide

  1. Sign up free at app.brevo.com
  2. Go to SMTP & API → SMTP tab
  3. Copy the Login and Password
  4. Click Brevo quick-setup button above
  5. Paste your login and password
Free limit: 300 emails/day. No credit card required.
